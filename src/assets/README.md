# Imágenes del Portfolio

Coloca aquí las imágenes 3D PNG:

- `hero-avatar.png` - Imagen principal del hero (avatar 3D que cargará Tincho)
- `about-avatar.png` - Segunda imagen 3D para la sección "Sobre mí"

Las imágenes deben ser PNG con fondo transparente, estilo 3D moderno.

Dimensiones recomendadas: 512x512px o 1024x1024px para mejor calidad.
