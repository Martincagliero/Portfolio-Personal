# Carga de media para Projects

Colocá aquí tus archivos para que se muestren en la sección `Projects` sin editar imports.

## 1) Office Auto Care
Ruta: `public/projects/office-auto-care/`

- `hero.jpg` → captura hero del sitio
- `desktop.jpg` → mockup desktop
- `mobile.jpg` → mockup mobile

## 2) Krautermeister
Ruta: `public/projects/krautermeister/`

- `hero.jpg` → captura hero del sitio
- `desktop.jpg` → mockup desktop
- `mobile.jpg` → mockup mobile

## 3) Krautermeister 3D (video)
Ruta: `public/projects/krautermeister-3d/`

- `showreel.mp4` → video principal del proyecto 3D
- `poster.jpg` → imagen de portada del video (opcional pero recomendado)

## Notas
- Podés usar `.png` o `.webp`, pero si cambiás extensión actualizá la ruta en `src/Portfolio.jsx`.
- Recomendado:
  - imágenes: 1920x1080 o similar
  - video: H.264 MP4, peso moderado para carga rápida
